Smart File Sorter
Smart File Sorter - это настольное приложение на Python (PyQt6), предназначенное для автоматического наведения порядка в ваших папках. Программа умеет распределять файлы по категориям на основе расширений или даты изменения.
🚀 Основные возможности
Сортировка по типам: Автоматическое распределение файлов по папкам (Изображения, Документы, Видео, Музыка и др.) на основе настраиваемых правил.

Сортировка по дате: Группировка файлов по месяцу и году их последнего изменения.

Гибкая настройка: Вы можете создавать свои категории, добавлять или удалять расширения прямо в интерфейсе приложения.

Сохранение настроек: Все ваши правила и последний выбранный путь сохраняются в файле config.json.

Безопасность: При совпадении имен программа автоматически переименовывает файлы, чтобы не перезаписать существующие данные.

Темная тема: Современный высококонтрастный интерфейс для комфортной работы.

🛠 Технологии
Python 3.x

PyQt6 — для создания графического интерфейса.


PyInstaller — для сборки в исполняемый файл .exe.

📦 Установка и запуск
Из исходного кода:
Установите зависимости:

Bash
pip install PyQt6
Запустите скрипт:

Bash
python script.py
Сборка в EXE:
Для создания автономного файла используйте PyInstaller. В репозитории уже есть настроенный файл script.spec.

Bash
pyinstaller script.spec
Готовый файл появится в папке dist/.

📁 Структура проекта
script.py — основной код приложения.


script.spec — конфигурация сборки для PyInstaller.

config.json — (создается при запуске) файл с вашими настройками категорий и путей.
